# How to run project

## STEP-1 : create .env file
## STEP-2 : run in terminal "composer install"
## STEP-3 : run in terminal "npm install" 
## STEP-4 : run in terminal "npm run dev"
## STEP-5 : run in terminal "php artisan serve"
